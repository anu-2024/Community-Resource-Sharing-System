
# Community Resource Sharing System

A platform where users donate or request unused household items.

## Tech Stack
Backend: Java Spring Boot  
Frontend: Streamlit  
Database: SQLite

## Run Backend
cd backend-java-springboot
mvn spring-boot:run

## Run Frontend
pip install -r requirements.txt
cd frontend-streamlit
streamlit run app.py

Backend API runs on:
http://localhost:8080
